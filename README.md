NYC Traffic Volume Analysis (Historical)

This project analyzes historical traffic volume data for New York City using publicly available datasets from NYC Open Data. 
The goal is to uncover trends in traffic patterns across different roadways, dates, and hours of the day.

Dataset

Source: NYC Traffic Volume Counts (Historical)

Description: Hourly traffic volume counts for NYC road segments, including road name, segment direction, and date.

Tools & Technologies

  Python

  Pandas for data wrangling

  Plotly and Matplotlib for visualization

  GeoPandas (optional for geospatial analysis)

  Google Colab for interactive development

  Tableau Public for dashboard visualization (optional)

Features

  Data cleaning and transformation: converts wide hourly data into a tidy long format

  Traffic trend analysis:

  Daily traffic volume trends

  Average traffic by hour

  Top busiest roadways

  Interactive visualizations with Plotly

  Export cleaned data for further use in Tableau Public

Impact

  This project demonstrates real-world data analytics and visualization skills using publicly available datasets, and highlights 
  the ability to handle large datasets, perform time-series analysis, and create interactive dashboards for insights.
